module

bookops_worldcat.authorize

</>

This module provides means to authenticate and obtain a WorldCat access token.

Classes
  • WorldcatAccessToken — Requests Worldcat access token. Authenticates and authorizes using Client Credentials Grant. Does not support Explicit Authorization Code and Refresh Token flows. Token with correctly bonded scopes can then be passed into a session of particular web service to authorize requests for resources. More on OCLC's web services authorization: https://www.oclc.org/developer/develop/authentication/oauth/client-credentials-grant.en.html</>

This module contains the set of bookops_worldcat exceptions.

Classes
  • BookopsWorldcatError — Base class for exceptions in this module.</>
  • WorldcatAuthorizationError — Exception raised when WorldCat access token is not obtained</>
  • WorldcatSessionError — Exception raised during WorlCat session</>
  • WorldcatRequestError — Exceptions raised on HTTP errors returned by web service</>
  • InvalidOclcNumber — Exception raised when an invalid OCLC record number is encountered</>
